You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- provide a num1= and num2= returns the resut Arguments num1=[String]&num2[String]
  • /github -- provide a repo has to be public
  • /area_of_triangle -- provide a doble type for parameters base= and height= return the area. Arguments base=[Double]&height=[Double]
  • /sentence - provide two strings with a sentence or word. Will return the strings in a sentence. Arguments str=[String]&str1=[String]

File Structure in www (you can use /file/www/FILENAME):

These are the link examples for the request above
  • http://18.222.182.39:9000/multiply?num1=9&num2=10
  • http://18.222.182.39:9000/github?query=users/rgarc168/repos
  • http://18.222.182.39:9000/github?query=users/amehlhase316/repos
  • http://18.222.182.39:9000/areaTriangle?base=5.3&height=7.2
  • http://18.222.182.39:9000/sentence?str=Hello&str1=World!
